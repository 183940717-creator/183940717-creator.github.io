# Hakuna 作品后台使用说明

后台地址：

https://hakunaworks.netlify.app/admin/

## 第一次使用前

需要在 Netlify 后台开启两项：

1. Identity
2. Git Gateway

开启后，用你的邮箱邀请自己成为后台用户，再从邀请邮件进入后台。

## 新增作品

进入后台后打开「作品集内容 / 作品列表与首页精选」：

1. 在「全部作品」里新增一条作品。
2. 填写作品 ID、排序编号、作品名称、年份和分类。
3. 上传默认封面。
4. 填写中文信息和英文信息。
5. 在「作品详情图片 / GIF / 视频」里按顺序上传作品内容。
6. 保存并发布。

作品详情页会自动生成，不需要新建 HTML 页面。

## 首页精选

「首页精选作品 ID」控制首页精选作品和顺序。

例如：

```text
pheagee
douyin-new-year
douyin-annual-report
aigc
```

要换首页精选作品，只需要改这里的 ID 顺序。

## 封面规则

首页和全部作品页会优先读取：

```text
首页作品封面/作品ID.jpg
首页作品封面/作品ID.png
首页作品封面/作品ID.gif
```

如果都没有，才会使用后台里上传的「默认封面」。

## 图片排列规则

作品详情图片会根据文件名编号自动排列。

例如：

```text
AI-001-1.jpg
AI-001-2.jpg
AI-001-3.jpg
AI-002.jpg
```

`001-1 / 001-2 / 001-3` 会排在同一行，`002` 会进入下一行。

如果想让某组图片渐隐轮播，在「渐隐轮播编号」里填写：

```text
001
```
