# Hakuna 作品更新说明

当前推荐使用「GitHub 文件式后台」，不再依赖 Netlify Identity / CMS 登录。

新增作品请看：

```text
GitHub添加作品说明.md
```

核心规则：上传图片到固定文件夹，然后修改 `content/projects.json`。

## 首页精选

`featuredProjectIds` 控制首页精选作品和顺序。

例如：

```json
"featuredProjectIds": [
  "pheagee",
  "douyin-new-year",
  "douyin-annual-report",
  "aigc"
]
```

要换首页精选作品，只需要改这里的 ID 顺序。

## 封面规则

首页和全部作品页会优先读取：

```text
首页作品封面/作品ID.jpg
首页作品封面/作品ID.png
首页作品封面/作品ID.gif
```

如果都没有，才会使用后台里上传的「默认封面」。

## 图片排列规则

作品详情图片会根据文件名编号自动排列。

例如：

```text
AI-001-1.jpg
AI-001-2.jpg
AI-001-3.jpg
AI-002.jpg
```

`001-1 / 001-2 / 001-3` 会排在同一行，`002` 会进入下一行。

如果想让某组图片渐隐轮播，在「渐隐轮播编号」里填写：

```text
001
```

本地保存前可以运行：

```bash
node tools/check-projects.js
```
